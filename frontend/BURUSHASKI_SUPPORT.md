# Burushaski Language Support

## Added for Hackathon Requirements

Burushaski (language code: 'bs') is now supported throughout the Project Barfani application.

### Language Information
- **Native to**: Hunza Valley, Gilgit-Baltistan, Northern Pakistan
- **Speakers**: Burusho people
- **Script**: Romanized Latin script (for this implementation)
- **Purpose**: Provide local language access for communities in glacier monitoring areas

### Implementation Details

**Supported Components:**
- ✅ Header (site title, connection status)
- ✅ Main Page (test mode, instructions)
- ✅ Dashboard (temperature, water level, seismic activity labels)
- ✅ Alert Panel (alerts, risk levels)
- ✅ Map (sensor nodes, locations)
- ✅ ML Insights (AI analytics)
- ✅ Test Panel (scenarios, testing modes)
- ✅ Manual Control (sensor controls)

**Language Selector:**
- Cycles through: English → Urdu → Burushaski → English
- Located in header top-right
- Globe icon button

### Usage
Users in Hunza Valley can now access the GLOF early warning system in their native Burushaski language, making it more accessible and culturally appropriate for the target community.
